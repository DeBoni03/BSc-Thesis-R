# BSc Thesis: Multidimensional approaches to the visual cortex's cytoarchitecture analysis in select mammals

### Abstract
Morphometric descriptors of visual cortex cells were recorded across twenty-three histological images from eighteen mammals, using an automated cell detection program. The descriptors span three informational domains: size, regularity, and neighbourhood density. This study aims to assess their characteristics in order to understand how evolutionary adaptation has shaped them, with regard to the species of origin and their phylogenetic relationships, through multivariate approaches such as cluster analysis and PERMANOVA. The thesis also provides an in-depth discussion of the data quality issues encountered, with particular attention to the significant challenge that preprocessing and data cleaning represent in this type of analysis. 

---

### The Challenge & Technical Focus
While the domain of this research is neurobiology, the project served as a rigorous testing ground for addressing real-world data complexities. Working with raw data generated by automated image detection systems presented significant hurdles, including severe noise, outliers, and dimensional complexity. 

Navigating these issues put my problem-solving abilities to the test, requiring the practical application of **Data Engineering**, **Unsupervised Machine Learning**, and **Multivariate Statistics** to build a robust and reliable analytical framework.

---

### Code Structure & Workflow

In this repository, the R code used in the study is shared alongside a synthetic dataset to safely test and reproduce the analytical pipeline. The workflow is divided into the following key sections:

**1. Preprocessing and Data Cleaning**
- Handling artifacts and anomalies generated by the automated cell detection software.
- Implementing robust scaling and normalization techniques to prepare multidimensional data for distance-based algorithms.

**2,3,4. Unsupervised Learning: Clustering Algorithms**
- **K-Means:** Partitioning data into discrete groups based on morphometric similarities.
- **Hierarchical Methods:** Exploring nested structures and relationships within the data using dendrograms.
- **DBSCAN:** Applying density-based spatial clustering to effectively isolate noise and identify non-linearly separable clusters.

**5. Model Evaluation (Mismatch and CVI Analysis)**
- Utilizing Cluster Validity Indices (CVIs) to mathematically evaluate and compare the performance and stability of different clustering algorithms without relying solely on visual interpretation.

**6. Statistical Validation (PERMANOVA)**
- Applying Permutational Multivariate Analysis of Variance to rigorously test the statistical significance of the identified groups across different mammalian species and phylogenetic relationships.
